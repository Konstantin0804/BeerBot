# BeerBot
Telegram bot for ordering beer from Zagovor Brewery.

For launching you need:
1. Define .env file in the root of the project with such fields:
  ENVIRONMENT=
  API_KEY=
  DB_PASSWORD=
  DB_USER=
  DB_SERVER=
  DEVELOPER_CHAT=
  PROD_CHAT=
2. pip install -r requirements.txt
3. Launch miklyx_bot.py 

Thats it.
